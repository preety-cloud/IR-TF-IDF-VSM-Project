The Influence of Data Characteristics and Usage on Direct Access File Organization .A Lowe, Thomas C. Memory utilization and retrieval time from direct access inverted files are investigated as a function of the data base, the demands on it, and a parameter which the system designer may control.. An analysis of the effects of data base characteristics and data base usage is also made for a linked list structure..
